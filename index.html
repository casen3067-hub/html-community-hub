📋 Changelog
v1.7 — Admin Power Tools Latest
  • Uploader name now shown on every game card
  • Admin-only games — hidden from regular players, toggle at upload or anytime
  • Ban screen now blocks the whole page instantly, not just on game open
  • Ban appeal form on the ban screen with admin Appeals tab
  • Admin can approve (auto-unban) or deny appeals
  • File editor & version history moved to Updates tab in Admin Dashboard
  • Admin Games tab: search, filter, archive, admin-only toggle, and delete
  • Admin-only games persist to MongoDB across restarts
v1.6 — Cover Images & Upload Fix
  • Added cover image upload with live preview
  • Upload button now shows ⏳ Uploading... while in progress
  • Toast notification on upload start and finish
  • Server errors now display instead of silently failing
v1.5 — Active Players Fix
  • Heartbeat now repeats every 10s to keep session alive
  • Players no longer disappear from admin dashboard after 15s
  • Heartbeat stops cleanly when game is closed
v1.4 — Bug Fix
  • Fixed SyntaxError: await outside async in admin dashboard
v1.3 — Admin Dashboard & Player Tracking
  • Admin dashboard with overview, players, games, analytics tabs
  • Active player tracking — see who's playing what in real time
  • Username + unique Player ID system on first visit
  • Uploader delete tokens saved in Settings → Tokens tab
  • Password-protected admin mode
v1.2 — Themes & Settings Menu
  • 9 color themes (Default, Snow, Lava, Nature, Ocean, Space, Cyberpunk, Sunset, Midnight)
  • Settings moved to its own full-page menu
  • Animated particle background
  • Favorites, Trending, Recent, My Games tabs
  • Game search bar
  • Fullscreen game mode
v1.1 — Persistent Storage & Categories
  • MongoDB Atlas + GridFS for persistent game storage
  • Games survive server restarts and redeploys
  • Category system (Action, Puzzle, Adventure, Strategy, Sports, Casual, Other)
  • Deployed to Render at doki-doki.onrender.com
v1.0 — Initial Release
  • Upload HTML files for the community to play
  • Browse and play games in an iframe viewer
  • Download games
  • Made with JavaScript (Node.js + Express), by Casen
